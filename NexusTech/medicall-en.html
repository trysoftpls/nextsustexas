Medicall Band

This device receives signals from the central unit and displays the rooms where help is requested. It also allows the user to select which room to assist. Like the central unit, it is equipped with an RYLR 998 for maximum efficiency, and its battery lasts up to 8 hours.

Component Description
ESP32 It processes decisions
RYLR998 Receives data from the Core-Station and transmits the response
OLED screen It allows you to see the interface
Pushbuttons They allow you to interact with the interface
Switch It allows to turn the device on or off
Battery Lasts for 8 hours
